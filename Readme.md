This repository contains the solutions for the Frontend Development Internship assignment.  
The project includes four tasks demonstrating responsive design, authentication, SPA behavior, API integration, and a fully functional Todo application.

## Tasks Overview

### Task 1: Responsive Webpage Development

Built using **HTML, CSS, JavaScript**
Fully responsive for:

- Desktop
- Tablet
- Mobile
  Clean layout and adaptive content

### Task 2: User Authentication Form

Login form with **client-side validation**
Validates:

- Email format
- Password length  
  On successful validation, a new user can be added to the application

---

### Task 3: Single Page Application (SPA)

Built using **React**
Fetches and displays data from a public API
Smooth page navigation and dynamic UI updates
Demonstrates:

- Components
- State management
- API fetching
- Rendering based on state

### Task 4: Todo Application

Responsive Todo App (mobile-first design)
Add, edit, delete tasks
Uses an API (or local storage) for data persistence
UI follows the provided design reference
